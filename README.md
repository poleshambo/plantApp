# plantApp
plants watering console tracker

type
showPlants() - to see all plants you have.

addPlant();  - to add plant. Ypu will be asked few questions.

deletePlant('name');  - to delete plant enter its name.

water('name') - to water a plant type this func.

savePlants() - to save it in internal browser memory.

loadPlants() - load them back.
